# Mood Tracker

## Описание проекта
Проект представляет собой приложение для отслеживания настроения пользователя. Пользователь может записывать своё настроение с комментариями, просматривать записи за разные даты, а также удалять их при необходимости. Приложение имеет графический интерфейс, предоставляющий доступ к календарю, истории записей и функции добавления новых записей.

## Реализуемый функционал
1. **Добавление новой записи**: Пользователь может выбрать настроение с помощью ползунка от -2 до 2 и добавить комментарий. Запись сохраняется с меткой времени.
2. **Просмотр записей**: Записи можно просматривать по дате в календаре. Каждая запись отображает время, настроение и комментарий.
3. **Удаление записей**: Пользователь может удалить записи, нажав на троеточие в соответствующей строке таблицы с историей.
4. **Интерфейс**: Взаимодействие с приложением осуществляется через графический интерфейс, основанный на библиотеке `Tkinter`.

## Архитектура
Проект состоит из нескольких файлов, каждый из которых выполняет свою функцию.

### Основные компоненты:
1. **MoodTracker** (`user.py`):
   - **Атрибуты**:
     - `file_path`: путь к файлу для хранения данных.
     - `entries`: список всех записей настроения.
   - **Методы**:
     - `load_entries_from_file()`: загрузка данных из файла.
     - `save_entries_to_file()`: сохранение данных в файл.
     - `add_entry()`: добавление новой записи.
     - `get_entries_by_date()`: получение записей по дате.
     - `delete_entry_by_id()`: удаление записи по id.
     - `update_entry()`: обновление записи.

2. **NewEntryWindow** (`new_entry.py`):
   - Отвечает за создание нового окна для добавления записи. Содержит ползунок для выбора настроения и поле для ввода комментария.
   - **Методы**:
     - `save_entry()`: сохраняет введенные данные в систему.

3. **CalendarWindow** (`calendar_view.py`):
   - Окно с календарем для выбора даты. По выбранной дате отображаются записи.
   - **Методы**:
     - `show_entries_for_selected_date()`: показывает записи за выбранную дату.

4. **HistoryWindow** (`history_view.py`):
   - Окно с историей всех записей. Отображает записи в виде таблицы с возможностью удаления записи.
   - **Методы**:
     - `load_entries()`: загружает записи в таблицу.
     - `on_delete_button_click()`: обрабатывает клик по кнопке удаления записи.
     - `confirm_delete()`: подтверждает удаление записи.

5. **GUI** (`gui.py`):
   - Главный интерфейс приложения. Отвечает за отображение кнопок для перехода в другие окна и запуск основной логики приложения.
