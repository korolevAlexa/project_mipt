# project_mipt
Описание проекта: Веб-приложение для трекинга настроения

Это приложение предназначено для отслеживания настроения пользователей в течение дня.
С помощью приложения пользователи могут записывать свое настроение, анализировать изменения в эмоциональном состоянии, а также использовать приложение как личный дневник.

Функциональность:
Регистрация и авторизация: Пользователи могут создать аккаунт и получить персонализированный доступ к своим данным.
Запись настроения: Возможность выбрать настроение из списка или задать уровень настроения на шкале. Пользователь также может добавить заметку или тег к записи.
Аналитика и графики: Приложение предоставляет графики и статистику, позволяющие просматривать динамику изменения настроения.
Ежедневные уведомления: Напоминания для пользователей о необходимости внести запись о настроении.
